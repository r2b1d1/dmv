Because sometimes all you need is to read a few lines of log.

dmv parses GENIVI 's dlt files exported from DLT Viewer or saved as offline logs.
It also parses dlt files converted to json using dlt2json tool.

Data is rendred to a table with filtering, sorting and searching options.

So go ahead and give it a try, load a file. or you can use the action button for more options.

+

Opens the menu.

home

Go back to home view.

table_chart

Go back to table view if it exists.

folder

Open a new dlt file.
For large files (> 30 MB) convert the file to json before loading it.

search

When in table view, this opens the search box.
Start typing to search the payload.
You can also filter on ecu, app, ctx, type or info,before searching the payload.
Search with /filter=value&filter=value/ search
e.g: /app=log&ctx=tes1/ hello
